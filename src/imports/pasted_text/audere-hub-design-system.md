Design a complete SaaS design system and shared UI component library for a platform called Audere Hub.

Audere Hub is the central operating system for business owners in financial services (financial advisers, accountants, mortgage brokers). The platform connects multiple AI-powered tools including:

• CoachAI – strategy and business coaching
• RoleAudit – role and capacity analysis
• FeeModeler – pricing and profitability modeling
• Marketos – AI marketing engine
• Finsight – financial performance analytics

The goal is to create a unified platform UI system so all applications share the same visual language and components.

Do NOT generate just a screen.
Generate a structured design system page with reusable components.

Visual Style

Professional SaaS platform similar to:

• Linear
• Stripe Dashboard
• Notion
• Vercel

Design principles:

• calm
• analytical
• minimal
• trustworthy
• data-focused

The interface should feel suitable for professional financial decision-making.

Colour System

Secondary
Slate greys and neutral backgrounds

Accent

Include:

• primary colours
• neutral scale

• background layers

Modern SaaS typography hierarchy.

• Section headers

Typography must prioritise data readability.
