# Shared UI

Shared UI components and design workspace for Audere apps (formerly the “Build this” bundle). The original Figma project: [Build-this](https://www.figma.com/design/vfhBDBs8bfJ3SUQQGSAO1k/Build-this).

## Running the code

Run `npm i` to install dependencies.

Run `npm run dev` to start the development server.

The publishable library is **`@audere/ui`** in `packages/ui`. To **publish** it and **link consumer apps** (npm, Vite, Lovable, local `npm link`), see **[docs/LINKING_APPS_TO_AUDERE_UI.md](docs/LINKING_APPS_TO_AUDERE_UI.md)**. Shell & navigation (hub rail + classic horizontal): **[docs/HUB_SIDEBAR_AND_NAV.md](docs/HUB_SIDEBAR_AND_NAV.md)**.
