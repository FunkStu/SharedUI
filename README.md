# Shared UI

Shared UI components and design workspace for Audere apps (formerly the “Build this” bundle). The original Figma project: [Build-this](https://www.figma.com/design/vfhBDBs8bfJ3SUQQGSAO1k/Build-this).

## Running the code

Run `npm i` to install dependencies.

Run `npm run dev` to start the development server.

The publishable library lives under `packages/ui` (`@audere/ui`).
